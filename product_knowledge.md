# Product knowledge

Reference facts about Hunter Industries / FX Luminaire products and terminology,
used to keep every generated episode accurate. Edit this from the app's
"Product knowledge" panel, or edit this file directly - either way, changes apply
to every episode generated afterward.

A general anti-hallucination rule (never invent specs not present in the source
material) is already built into the app and doesn't depend on this file. Use this
file for things that are always true and worth reminding the model of - what a
product/system actually is, common mix-ups to avoid, correct pronunciations, etc.

Keep entries short and certain. Wrong information here is worse than no
information - it will get treated as fact.
